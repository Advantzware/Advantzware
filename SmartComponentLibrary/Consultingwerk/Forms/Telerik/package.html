The Consultingwerk.Forms.Telerik package contains specialized GUI for .NET Form base classes based on the Telerik UI for WinForms RadForm or RadRibbonForm. The classes do extent those forms and provide further framework related functionality.

Class name Description
BaseRadForm The main additional functionality of this class is to store the last position and size in the Windows registry or a custom ISettingsService implementation. This Form supports Telerik Styling but is not intended to by used with the RadRibbonBar.
BaseRadForm The main additional functionality of this class is to store the last position and size in the Windows registry or a custom ISettingsService implementation. This Form supports Telerik Styling and provides an RadRibbonBar instance.

See the MSDN reference for the System.Windows.Forms.Form class for further information.
