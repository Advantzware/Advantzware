The Consultingwerk.Studio.Desktop package contains the SmartComponent Library Desktop. This is a launcher Application for various tools of the SmartComponent Library framework. The Desktop application will make it easier for us to roll out "shortcuts" to launching new tools or resources to our customers. Customers will only need to create a single desktop shortcut to the "SmartComponent Library Desktop" and we can deploy additional items to this desktop.



See the Confluence page for more information.
