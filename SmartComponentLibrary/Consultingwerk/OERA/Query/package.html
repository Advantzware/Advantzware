The Consultingwerk.OERA.Query package contains the core query manipulation used by the functionality used by the DataAccess objects. The DataAccess class implements the IDSQueryStringSource interface which contains callback methods required by the Query manipulation routine.

Class name Description
IDSQueryStringSource Interface implemented by the DataAccess classes that allows callbacks from the DSQueryString class
DSQueryString Query manipulation class for class based Data Access objects
DSQueryStringProcedural Query manipulation class for procedural Data Access objects
DSQueryString Core query manipulation routines.
