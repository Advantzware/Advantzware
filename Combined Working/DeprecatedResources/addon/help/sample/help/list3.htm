List Request (Page 3 - Output Selection)

This page allows for the output destination to be selected.

Printer allows one of two options. If left as is when the Finish button is clicked the List process will use the operating systems default Printer Dialog screen for printer selection. If the Spool toggle box is checked the List process prompts for a date and time to run this List using the Parameters entered on Page 1 and Page 2. The Spool option only works if the System Administator has activated the Spool Monitor module to run in background.

A short cut exists which bypasses the selection of the Printer option and clicking the Finish button, simply click the Print icon to the left of the Printer, Screen and Disk File option and the List process will immediately produce the List to the default printer selected with the operating system setup.

Screen will cause the list be produced within an Editor for easy viewing, click this icon to view Screen Viewer Instructions.

Disk File prompts for a file name and location using the operating system's file dialog screen when this option is selected and the Finish button is clicked.

Finish button produces the List based on the Parameters entered on List Pages 1 & 2 and also on the values entered here..

Back button returns the List to Page 2 - Show Options.

Cancel at any time during this process to return to calling module.

click this icon to see List Page 2- Show Options Instrunctions.
