'To track a user's comings and goings from system modules requires setting the Track Usage toggle box. The results can be viewed by running the Track Usage List under System: System Listing.

If use of custom colors and fonts is desired, the Use Defined Colors/Fonts toggle boxes must be set.

The Developer toggle box should be used only by the login ID of the software developer. Use of this toggle box can have undesired results for those users not trained to do so.

An individual can set what image appears on the Main Menu screen by placing an image Bit Map name in the Image File Name field.
