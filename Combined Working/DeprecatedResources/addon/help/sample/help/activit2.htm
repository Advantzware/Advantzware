After entering a Card Number or a Registry value, the Valid Names field will be populated with those names associated with either the Card Number or Registry if a value is entered. Simply select from the Valid Names combo box object that name which is appropriate. If the name desired does not appear within the Valid Names selection, then that name must be entered either in the Blind Code Cards or Registry/Master Key modules. After entered a valid name in either module, return to this screen and the name added will appear as a valid selection.
