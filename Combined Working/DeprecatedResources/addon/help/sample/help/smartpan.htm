Smart Objects Panel

This panel appears at the top of most every window. Simply mouse click on desired item or press associated function key.


Search (F2) - pops up a search browser allowing record searches using test contained within data fields.

List (F3) - calls a default listing associated with window currently open.

Notes (F4) - allows notes entry to the currently selected record.

Misc Fields (F5) - allows Misc Fields data to be maintained.

Browser (F6) - places page 1 browser into focus.

Viewer (F7) - places page 2 viewer into focus.
