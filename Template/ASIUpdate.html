MANUFACTURING & DISTRIBUTION SOFTWARE FOR PACKAGING









Thank you for your interest in Upgrading your Advantzware software installation. 

Please follow the instructions on this page to download and install your Advantzware updates.

(NOTE: You will need to run the update while logged into the Advantzware server, and you will need an installation password.
You can get this from an ASI Support representative.)

To begin, click this link to download the "zipped" patch file.  By default, this will be placed in your Downloads folder on the server.  When the download is complete, follow the steps listed below:

1. Open Windows Explorer and locate the Downloads folder
2. Find the Patch file you just downloaded
3. COPY the file to the Updates directory of your Advantzware installation (usually N:\Updates)
4. Right-click the Patch file, find the 7-zip option, and choose "Extract Here"
5. Double-click the file named "StartUpdate.bat" (or -sometimes- just "StartUpdate")

At this point, the patch will begin the update process. 

When prompted, enter the update User ID and Password provided by ASI Support.

If you have multiple environments (Prod and Test, for example), choose the environment you wish to upgrade and press the Start Update button.  (For customers with a single environment, the process will begin automatically.)
