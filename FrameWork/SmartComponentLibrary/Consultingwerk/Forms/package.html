The Consultingwerk.Forms package contains specialized GUI for .NET Form base classes. The classes do extent Progress.Windows.Form / System.Windows.Forms.Form and provide further framework related functionality.

Class name Description
BaseForm The main additional functionality of this class is to store the last position and size in the Windows registry or a custom ISettingsService implementation.
MDIContainer The MDIContainer class extends the BaseForm and provides additional functionality for the MDIContainer of an application, such as keeping the current active MDI Child's title added to the MDIContainer's title.

See the MSDN reference for the System.Windows.Forms.Form class for further information.
