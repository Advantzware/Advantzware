List Request (Page 2 - Show Options)

This page allows for the inclusion of additional information. NOTE: not all of these options are available on all Lists.

Show Notes option when selected allows for the inclusion of the notes related to selected records, only those note records which have the Viewed flag set, or only those note records which do not have the Viewed flag set. Click this icon for further instructions of Notes.

Show Selection Parameters will cause the list to produce a first page showing the Parameter values from the first page as well as those entered on this page.

Show Misc Field Values will add any Misc Field labels and value if they exist for listed records.

Show Addresses adds any address information if applicable for this List.

Show Phones functions like addresses in that it only will print if the type of records listed have phone information attached.

Next button continues the List process to Page 3 - Output Selection.

Back button returns the List to Page 1 - List Parameters.

Cancel at any time during this process to return to calling module.
