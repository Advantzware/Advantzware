Transaction Panel

This panel appears in association with most viewer objects, and allows record data manipulation. Access to each buttons functionality is based on security permissions implemented by the System's Administrator.

Update - allows access to currently selected record and changes the button's label to Save.

Save - saves currently displayed values entered from an Update, Add or Copy action.

Reset - is active when an Update, Add or Copy action is done and will restore a records original values since the currently selected record was last Saved.

Add - allows entry on a new record's data into a viewer's associated table and changes the Update button's label to Save.

Copy - creates a new record and sets all of the enabled fields values to those of the currently displayed record and changes the Update button's label to Save.

Delete - after prompting for confirmation of deletion action, removes the currently selected record from the database.

Cancel - will cancel any action taken and restores the records values as they were before cancelled action taken.

When using the Update, Add or Copy actions may reveal the presence of an F1 icon(s) indicating the field has Lookup capabilities, otherwords, rather then enter a value manually, press the F1 key on the keyboard to see a pop-up of valid values for the currently selected field.
