Lookups

The lookup pop-up functions exactly like the browser objects.

The displayed records can be sorted by changing the radio object value associated with the label 'By:'. The sample screen has sort values 1 Title and 2 Program, simply select which radio value is desired or type ALT-1 or ALT-2.

When the browser object has focus (portion which contains record data value), navigation can be done by typing a desired value which begins trying to match a fields value from left to right. This is referred to as an Auto Find. If you wanted to find the record with Program field value 'audit_' simply type an 'a', then 'u', then 'd' and the browser will begin displaying only those records that first begin with an 'a', then 'u', then 'd', etc. If the sort value was 2 Program, the same functionality would display only those records which the Title field value begins with the values entered.

A faster method of doing the same functionality can be done by typing a beginning value into the field found below the Clear Find button and hitting Enter. This causes the browser to filter the desired records once with all of the entered values instead of key stroke by key stoke. This is helpful for those tables that contain a very large amount of records.

At any point click the Clear Find button or type ALT-F to erase any previously entered Auto Find values and return the browsers records to their original values.

Either double mouse click the desired record or click the OK button to select the currenly selected record once found. Upon closing this window, the selected record will now be highlighted or displayed as the current record in the window from where the Search function was originally called.
