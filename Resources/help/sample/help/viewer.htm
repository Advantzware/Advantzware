Viewer Screen Instructions

Click on any of the shown panels for further information.

Navigate by using your mouse to pull down menu items or clicking on any of the Smart Object Panel icons . Change pages by using the folder tabs.

Use the Navigation Panel to move forward and backward in relation to currently selected record.

Use the Transaction Panel to manipulate the currently selected record's data.

The presence of an F1 icon indicates the field has Lookup capabilities, otherwords, rather then enter a value manually, press the F1 key on the keyboard to see a pop-up of valid values for the currently selected field.
