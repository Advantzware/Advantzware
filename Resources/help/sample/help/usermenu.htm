User's Menu Structure

Use of this module allows individual users to customize their menu selections. Select any Menu or Program Item from the right side selection list boxes and either double mouse click the selected item or use the Add Item buttons to move the selections to the left side selection box.

The left selection box shows how the custom menu structure will appear once Saved. Use the buttons below this structure to manipulate the individual items for desired results. At the top of this screen appears the pull down menus showing how the pull downs will appear when operational.

lick the Build Menu button to move any changes to this pull down menu. The Build Menu button does not make the changes permanent.

The Default Menu button will reset the menu structure to be identical to the systems default menu structure.

The Reset Menu button will restore the menu structure to at structure when last saved.

No changes will become operational until the Save Menu button is selected.

WARNING: If changes are made to a system's default menu structure, and a custom menu is in use, those changes will not be reflected within the custom menu structure. Manual manipulation through this module is needed in order to apply the default menu structure changes.
