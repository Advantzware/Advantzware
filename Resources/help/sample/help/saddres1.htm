Enter Search text to the right of the icon, then click this icon to intiate record selection based on the Search text entered. Only those record which contain the Search text will be displayed. Select the desired record and click the icon to open the program module which contains the parent record that the selected record is attached. The newly open program will be positioned within the browser object highlighting the parent record.
