ScreenViewer

This screen shows the results of a List Request processed for screen viewing.

Open button gives viewing access to this or any previously run Lists to the screen.

Print button allows the currently selected List Request to be sent to any system printer.

Delete will remove the currently select List Request, note: user's are responsible for removing List Request run to the screen.

Save As allows users to copy the current selection to another disk file name.

Search button will highlight the text within the current selection based on the text entered in the Search field.

Next and Previous will be enabled when text exists within the Search field allowing a user to find multiple occurances of the Search text with the current selection.

Font button allows a user to change the font size used to display the current selection. This will not effect the font size of the selection if sent to a system printer.
