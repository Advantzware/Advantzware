This module is the heart and sole of this system. All programs with the system have an entry within this table.

Items with Program names ending with a period are executable, those without a period of simply menu items, i.e. File, User, Codes, etc.

The Title field value is used on the Main Menu pull down menus.

Directory value indicates the physical locatation on disk of the executable module.

Misc Fields Group defines which custom user defined field group is attached to this module. (advanced training is required for use of this functionality).

Check the Menu Item toggle box if this entry can appear on the Main Menu pull down menu.

From the Main Menu, type CTRL-P to access a quick pop-up feature showing those entries which have the Pop-Up toggle box checked.

Run Persistent means that this module is executed in memory similiar to an old style TSR type program.

Track Usage allow the system administrator to monitor which users access this module.

Can Run editor box shows a comma delimited list of those users and/or user groups with access to run this module.

Can Create editor box shows a comma delimited list of those users and/or user groups that can add or copy new records to the database.

Can Update editor box shows a comma delimited list of those users and/or user groups which can manipulate existing record data.

Can Delete editor box shows a comma delimited list of those users and/or user groups that can delete an existing record from the database.
